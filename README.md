# Project Management System

A full-stack MERN application for managing projects, users and tasks with role-based access.

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features

- Admin, Manager, Employee roles
- User management
- Project creation
- Task assignment
- Dashboard analytics
- Reports generation
